Luas Segitiga

Rumus Luas Segitiga Yaitu:

L = 1/2 x a x t

    Di mana:
  • L = Luas
  • a = Panjang Alas
  • t = Tinggi

Hitung Luas Segitiga

  • L = 1/2 x a x t
  • L = 1/2 x 0 x 0
  • L = 0

Keliling Segitiga

Rumus Keliling Segitiga Yaitu:

K = S1 + S2 + S3

    Di mana:
  • K = Keliling
  • S1 = Sisi A
  • S2 = Sisi B
  • S3 = Sisi C

Hitung Keliling Segitiga

  • K = S1 + S2 + S3
  • K = 0 + 0 + 0
  • K = 0
